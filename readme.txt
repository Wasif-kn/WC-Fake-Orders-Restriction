Restrict fake or spam orders in WooCommerce based on email, phone number, or IP address.

== Description ==
WC Fake Orders Restriction is a plugin designed to prevent orders from being placed in WooCommerce based on specific criteria such as email address, phone number, or IP address. This helps enhance security and mitigate potential fraudulent or fake orders on your WooCommerce store.

== Installation ==
1. Upload the `wc-fake-orders-restriction` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure your order restrictions via the 'Restrictions' menu in the WordPress admin dashboard.

== Features ==
- Restrict orders based on email, phone number, or IP address.
- Provides an intuitive admin interface to manage and view restrictions.
- Prevents potential fraudulent or fake orders in your WooCommerce store.
- Compatible with WordPress and WooCommerce versions up to 5.9.

== Frequently Asked Questions ==
= How do I add a restriction? =
Navigate to the 'Restrictions' menu in the WordPress admin dashboard. You can add restrictions by selecting the restriction type (Email, Phone Number, or IP Address) and entering the specific value you want to restrict.

= Can I view existing restrictions? =
Yes, the plugin offers a table in the 'Restrictions' menu that displays existing restrictions, including their IDs, types, and values.
